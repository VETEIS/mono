# PWA Icon Generation Instructions

To create the PWA icons, you need to generate PNG files:

1. **icon-192x192.png** - 192x192 pixels
2. **icon-512x512.png** - 512x512 pixels

These icons should:
- Have a dark background (#1C1C1E)
- Feature a yellow "M" or MONO logo (#FCD34D)
- Be square PNG files

## Quick Options:

### Option 1: Online Tool
1. Create or find an icon image
2. Use an online tool like https://realfavicongenerator.net/
3. Upload and download the generated icons

### Option 2: Image Editor
1. Create 192x192 and 512x512 images in any image editor
2. Use your app logo or design
3. Save as PNG files in the `public` folder

### Option 3: Use the SVG template
1. Open `icon-template.svg` in a design tool
2. Export as PNG in both sizes
3. Place in the `public` folder

Replace the placeholder files with your actual icons before deploying.
